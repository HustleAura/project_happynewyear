App workflow
- The app starts with the auth page or home page
- If auth page then the user starts with the home page after authentication
- the home page contains list view of entries
- a drawer for profile and a floating action button for new entry.
- on clicking the floating button we open a new entry page.

Screens
- Register 
- Sign In page
- Landing page (Just a list view of diary entries and access to everywhere)
- Diary Entry page (page for writing and formatting controls)
- user profile drawer
